Mini TaskPhase

This is a mini TaskPhase, where the participants will be covering some basic topics in Deep Learning.
Upon Applying, you can expect an email within 2-3 days including the schedule and setup information required to get started. Considering the date of initiation, the mini-TaskPhase will be divided into two parallel task courses, one for juniors and other for seniors.
You are free to work on whichever you like based on your skills but do keep in mind that you should only go for the tasks that match your skills.

January First Week

DL Projects Management & Technical Documentation

This is an important step as it involves introducing the members to the best practices when it comes to managing a software project. This involves learning the use of various industry tools, learning how to debug code, and learning how to write proper documentations which also work as a proof of work. All work performed under the department would be accompanied by a public repository on GitHub and a blog article if needed be.

The approach used towards learning is to first tackle the terminology then building an intuition and then understanding the mathematics behind the concept after which we’d move to programming.

Implementing Guided Projects

This step involves getting started with Deep Learning and Reinforcement Learning by implementing various popular projects. The purpose of this step is to build confidence and implement existing algorithms, helping the members explore the extent of the subject.

Advanced Topics within Deep Learning

This is where we explore advanced concepts behind modern deep learning models.

Understanding and Implementing Research Papers

The purpose of this step is to understand cutting-edge Deep Learning research, which is incredibly important for Ai Research.

Enigma Ai Research Project

This step is supposed to run parallel to all other club activities and exclusive to the project’s team. The project must be an original implementation of an idea. Seniors or Juniors can be a part of this project depending on their skills.
